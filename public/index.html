 Basic Cell Set (BCS) - FSMF or FBBC

 up to 6x10MHz / 2x2MIMO

 up to 3x10MHz / 4TX/4RX

 up to 3x20MHz / 2x2MIMO

 4TX/4RX = 4x2 MIMO, 4X4MIMO or/and 4RX

 Extended Cell Set (ECS) - FSMF + FBBC

 up to 3x20MHz / 4TX/4RX

 Reduced Cell Set (RRCS) - FSMF*

 up to 3x10MHz / 2x2MIMO with CoMP

 up to 4x10MHz / 2x2MIMO without CoMP

 *FSMF capacity is shared for LTE and GSM
